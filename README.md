## 内容摘要

本书以大数据应用方面常用的语言Python为基础，从网络爬虫的实现原理入手，逐步引领读者进入网络爬虫的世界。在各类爬虫框架中，将Scrapy作为轴心，从多个维度揭开爬虫技术的面纱。例如，爬取规则的制定技巧，设计高速爬虫，如何让爬虫更“聪明”地获取数据，将海量数据进行分布式存储的技术，设计具有高隐匿性的爬虫，大规模、高并发的分布式爬虫技术。
